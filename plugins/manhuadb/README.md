# ManhuaDB Plugin

This is Mango plugin for [ManhuaDB](https://www.manhuadb.com/).

**Attention**

* Only show the results in the first page when search.

* It is not easy to get the number of chapter pages before starting the download, so in the chapter list the number of pages for all chapters is set to 1000. But in the download manager the number of chapter pages is correct.

Maintained by [@Ftbom](https://github.com/Ftbom).
